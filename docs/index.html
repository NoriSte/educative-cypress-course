React Testing Course

ReactJSDay 2019 Testing Course

This is the reference repository with all the contents and the examples of the "React Testing 101: component, integration, and end-to-end testing" course for the ReactJSDay 2019 conference.

The repository aims to be consumed through the compiled GitBook. The course starts from the peak of the testing pyramid and goes down instead of following the "standard" testing path, you can read here the reason why. There are event some introductory slides but they're useful for the teacher only, they're a summary of the contents up to the coding session start.

Additional requisites to run the project and the tests on your local machine:

  • Node.js 10+
  • MongoDB
  • Docker
